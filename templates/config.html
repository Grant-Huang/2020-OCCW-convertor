规则说明
变量说明：
  • OCCW编码 - 用户编码去掉"-L"和"-R"后的结果
  • 花色 - 从Door Color字段获取的颜色代码
  • User Code - 用户编码
当前规则：
  • Cabinet: OCCW编码去掉"-L"和"-R"，SKU=OCCW编码&"-PLY-"&花色
  • Hardware: SKU="HW-"&User Code
  • Accessory: SKU=花色&"-"&User Code
  • 默认: SKU=花色&"-"&User Code
提示：修改规则后，需要重新上传报价单PDF才能看到新的SKU生成结果。
